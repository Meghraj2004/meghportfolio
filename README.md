# 🚀 Meghraj's Developer Portfolio

Welcome to the repository for my personal developer portfolio! This project showcases my skills, experience, projects, and achievements as a Full Stack Developer. It's built with modern web technologies to be fast, responsive, and visually stunning.

## ✨ Features

- **Modern UI/UX:** A clean, dark-themed aesthetic with a polished look and feel.
- **Smooth Animations:** Integrated with `framer-motion` for fluid page transitions, scroll animations, and interactive elements.
- **Projects Showcase:** Detailed project cards with SVG illustrations, technical stacks, descriptions, and direct links to live demos and repositories.
- **Achievements & Certifications:** A dedicated section displaying hackathon wins (like Avinya and Phoenix 2026) and professional certifications (AWS, GCP).
- **Fully Responsive:** Mobile-first approach using `Tailwind CSS` to ensure perfect rendering on practically any device.

## 🛠️ Technology Stack

- **Frontend Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** SVG and Heroicons

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have Node.js installed on your computer.
* [Node.js](https://nodejs.org/en/) (v16.x or newer recommended)
* `npm` or `yarn`

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Megharaj170804/portfolio.git
   cd PortfolioForge
   ```

2. **Install the dependencies:**
   ```bash
   # If you're using npm
   npm install

   # If you're using yarn
   yarn install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` (or the port specified in your console) to view the portfolio.
